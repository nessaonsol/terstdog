Introduction to Moonpad

Welcome to Moonpad, the first native launchpad built specifically for the Moonshot ecosystem. This documentation will guide you through everything you need to know to successfully launch your token project.

What is Moonpad?

Moonpad is a decentralized token launchpad that leverages the power of Moonshot's blockchain infrastructure. We provide a comprehensive platform for:

  • Token creation and deployment
  • Fundraising and initial distribution
  • Community building and governance
  • Analytics and project management

Key Features

Our platform offers several advantages over traditional launchpads:

Moonshot Integration

Built natively on Moonshot, we offer unparalleled transaction speeds and minimal fees, making token launches accessible to everyone.

Security

All smart contracts are audited by leading security firms, and we implement multi-signature controls for enhanced security.

Note: All projects undergo a thorough vetting process before being approved for launch on our platform.

Quick Start Guide

Get your token launched in just a few steps:

Step 1: Connect Your Wallet

Connect your Moonshot-compatible wallet to the platform. We support:

  • MoonWallet
  • MetaMask (with Moonshot network)
  • WalletConnect

Step 2: Create Your Token

Use our token creation wizard to define your token parameters:

{ "name": "Your Token Name", "symbol": "YTN", "totalSupply": "1000000", "decimals": 18, "mintable": false }

Step 3: Configure Launch Parameters

Set up your token sale parameters including pricing, duration, and distribution rules.

Warning: Once deployed, token parameters cannot be changed. Please review carefully before proceeding.

Launch Requirements

Before launching on Moonpad, ensure you meet these requirements:

Technical Requirements

  • Valid Moonshot wallet with sufficient balance for deployment fees
  • Completed KYC verification
  • Project whitepaper and documentation
  • Smart contract audit (for advanced projects)

Financial Requirements

  • Minimum launch fee: 0.1 MOON
  • Platform fee: 2% of funds raised
  • Gas fees for deployment

Token Creation

Learn how to create and customize your token using our platform.

Token Standards

Moonpad supports the following token standards:

  • MOON-20 (Standard fungible tokens)
  • MOON-721 (Non-fungible tokens)
  • MOON-1155 (Multi-token standard)

Creating Your Token

Use our intuitive interface to define your token's properties:

// Example token configuration const tokenConfig = { name: "MyToken", symbol: "MTK", totalSupply: ethers.utils.parseEther("1000000"), decimals: 18, burnable: true, pausable: true };

API Overview

Integrate Moonpad's functionality into your applications using our REST API.

Base URL

https://api.moonpad.io/v1/

Authentication

All API requests require authentication using API keys. Include your API key in the header:

Authorization: Bearer YOUR_API_KEY Content-Type: application/json

Rate Limits

API requests are limited to 1000 requests per hour per API key.
